# Akthos-Idle

Akthos-Idle is an experimental idle RPG for Android.

## Building
1. Install the Android SDK and ensure `local.properties` points to it (`sdk.dir=/path/to/sdk`).
2. Run `./gradlew build` to compile the app.
3. Run `./gradlew test` to execute unit tests.

## Features
- Basic combat math utilities
- Tick service that awards offline progress
- Simple loot-table rolling

## Contributing
Contributions are welcome! Please run the test suite before opening a pull request.
